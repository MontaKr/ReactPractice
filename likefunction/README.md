# 부모,자식 컴포넌트의 state를 넘기는 기능

부모 컴포넌트에서 `isNum` state를 자식 컴포넌트에 넘겨서 자식 컴포넌트에서 `isNum` state를 사용가능 \
자식 컴포넌트의 `isValue` state는 부모 컴포넌트에서 `passValue` 함수를 선언해서 함수를 통해 state를 사용 가능

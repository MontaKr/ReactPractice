# slick-slider 라이브러리를 이용한 슬라이더 기능

slick-slider 기능을 이용하여 슬라이더 기능 구현 \
다양한 기능을 가진 슬라이더 구현 용이. 마우스가 호버될 때 화살표 생성. 
setting에 `beforeChange`, `afterChange`를 추가하여 현재 슬라이더의 번호 표시

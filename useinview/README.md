# react-intersection-observer 라이브러리를 이용하여 뷰포트가 해당 태그가 뷰포트에 들어와 있는 지 여부 확인

useInView를 활용하여 타겟 태그의 속성에 ref를 만들고 inView에 태그가 뷰포트에 들어가는 지 여부에 따라 boolean값이 변경
